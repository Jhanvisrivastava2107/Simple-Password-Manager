#Project Statement

#Problem Statement
Poeple often need a quick way to store their simple passwords without using complex tools. This project provides an extremely simple password manager for beginners to understand file handling

#Scope of the project
-Stores passwords in a local text file
-Add and view passwords entries
-Provide a simple CLI interface
-Basic validation and Error Handling
-Focus on learning fundamental programming concepts

#Target Users
- Student learning basic python
- Bgeinners practicing file handling
- Users who want a minimal offline password tracker

#High level Features
- Menu Driven Interface
- Add password entries
- View saved passwords
- Text file based storage
- Error handlig for missing files
