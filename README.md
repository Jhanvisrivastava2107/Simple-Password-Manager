# Simple-Password-Manager
This is a very basic password manager written in Python which is beginner friendly to create and would help you store your passwords. This is designed for beginners and academic submissions.

#Features
-Add new passwords
-View old passwords
-Simple text file
-Basic error handling for missing files

#Tools Used
-Python
-Git
-Text file Storage

#Steps to Install and Run the Project
-Clone or Download the project folder
-Open a terminal in the project directory
-Run the program 

#Instructions for testing
-Start the program and choose "Add Password"
-Enter random service names and passwords
-Choose "View Passwords" to confirm they were saved
-Open 'passwords.txt' to verify whether the entries exists
-Test with an empty or deleted file
-Test invalid inputs
